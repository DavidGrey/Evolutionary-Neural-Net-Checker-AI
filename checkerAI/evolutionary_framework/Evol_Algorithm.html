Class Evol_Algorithm

Class Evol_Algorithm

public class Evol_Algorithm

Evolves a population of Evol_Objects. It can be quickly used by creating an instance of an Evol_Object and using that as a parameter when creating an instance of an Evol_Algorithm and then calling the search method.

Method summary
void crossover_two_children()
           By default, crossover is implemented by finding a pair of parents for each for each child element created. Calling this method will cause each pair of parents chosen to create two child elements.
void enable_crossover(int percent_crossover)
           Calling allows a percent_crossover% (default 0) chance of crossover for each child. If the child is not created through crossover, it is created by mutation.Crossover and mutation occuring simulataneously is current not supported.
void enable_print_population()
           Calling this method enables printing each Eval_Object's fitness values each generation.
int generations()
           After a search is performed, this may be called to return the number of generations that the search took. This can be used to determine whether or not your search failed (if applicable).
void max_solutions(int max_num_solutions)
           Calling this function enables this optional stopping condition. It stops after a certain number of potential solutions have been viewed. This is calculated by population size*number of generations. This gives more control than the max_generations stopping condition.
int popsize()
           Returns the population size that this object uses to perform an evolutionary search.
void recompute_elitists()
           If this function is called, the fitness of the elitist elements will be recalculated for each generation.
PrioQueue search()
           Creates a random population of Evol_Objects and evolves them until a stopping criteria is met.
void set_selection_type(int selection_type)
           Sets the selection operator (default roulette selection).
           1. Roulette selection. (Currently no other selection operators are available)

public Evol_Algorithm(Evol_Object rt, int population_size, int stop_val, int max_gen, int num_elite)

           Creates an instance of Evol_Algorithm.

           Parameters

           rt - An instance of an Evol_Object which contains a representation of the problem that you wish to solve.
           population_size - The population size to use while executing the evolutionary algorithm.
           stop_val - Stop the search when the fittest element's fitness is equal to this value. -1 disables this stopping condition.
           max_gen - Stop the search after this many generations.
           num_elite - Number of elitist elements.

Method detail

search

public PrioQueue search()
1. Creates an initial randomized population of Evol_Objects.

2. While a stopping criteria is not met:
Generates a new population using the roulette wheel selection technique and the mutation(s) specified in Evol_Object's mutate() method.
3. Return the final population of Evol_Objects as a PrioQueue object.
