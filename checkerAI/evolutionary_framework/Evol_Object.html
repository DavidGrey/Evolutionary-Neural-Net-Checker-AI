Interface Evol_Object

Interface Evol_Object

public interface Evol_Object

Used with Evol_Algorithm class which evolves a population of Evol_Objects. Specifies an instance of a solution to your problem and how to mutate from this instance to a child instance. Crossover is not currently supported.

To successfully use this package you must implement the abstract classes and override the clone() method. The class should contain a representation of your problem and an instance of your solution. For example, if you were implementing a TSP class this class would store the distance matrix (representation of problem) and one permutation of cities (instance of solution).

Method summary
abstract Object clone()
           Copy all attributes of this object to a new Evol_Object instance and return the clone.
abstract void compute_fitness()
           Calculates and stores the fitness value for this element.
abstract Evol_Object crossover(Evol_Object parent2)
          
This method may be implemented with a semi-empty body ( {return(this);} ) if crossover is not required.           
1. Create a (child) clone of this.           
2. Make the child the result of crossover between parent2 and this.           
3. Return the child.
abstract int get_fitness()
           Return the stored fitness value.
abstract Evol_Object head_to_head(Evol_Object contender)
           This method is used with tournament selection. Let contender and this compete and return the winner. If tournament selection is not required, it may be implemented as a semi-empty method such as { return(this); } because it will never be called.
abstract void initialize_representation()
           This method may be implemented with an empty body ( {} ). This method initializes a problem representation. An example would be to create a random distance matrix for the TSP or a random valid puzzle for the N-puzzle. Implementing this method allows EvolParamOptimizer to optimize for the problem in general, not a specific instance of it.
abstract boolean is_minimization ()
           Return true if this is a minimization problem, false if this is a maximization problem.
abstract Evol_Object mutate()
           1. Create a clone of this.
           2. Perform a mutation on the clone (child)
           3. Return the newly mutated child
abstract Evol_Object randomize()
           1. Create a clone of this.
           2. Create a random solution for the clone.
           3. Return the clone.
